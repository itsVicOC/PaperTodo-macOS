## 版本记录

本仓库只记录 PaperTodo macOS 版的用户可见变化。Windows 版历史、下载与后续更新请查看原作者仓库：https://github.com/snownico0722/PaperTodo

### Unreleased

### v2.0rc2

- **原生 macOS 版本**：新增基于 AppKit 的菜单栏应用，无主窗口、无 Dock 图标，提供独立桌面纸片体验。
- **macOS 仓库定位**：本仓库收敛为只维护 macOS 版，Windows 版本说明与下载统一指向原作者仓库。
- **macOS 发布流程**：Release 流程只构建和发布 macOS 未公证 `.app.zip`，并保留 SHA256 与 Sigstore 校验信息。
- **macOS 正式文案**：应用菜单和说明文档移除“原型”表述，改为正式的 macOS 版定位。
- **待办纸**：支持勾选、编辑、删除、清理已完成、回车新建、空行退格删除、多行粘贴拆分、拖拽排序和拖到底部删除。
- **待办撤销/重做**：支持勾选、增删、清理、拖拽排序、多行粘贴和关联笔记变更回退。
- **笔记纸**：支持普通文本编辑、轻量 Markdown 显示、正文缩放、格式快捷键和右键格式菜单。
- **Markdown 显示**：支持标题、列表、任务列表、引用、分割线、行内代码、代码块、链接、常见行内强调样式和少量单行行内 HTML 标签。
- **Markdown 链接交互**：支持在笔记编辑区按住 Command 点击打开 Markdown 链接，并可从链接右键菜单打开。
- **外部编辑同步**：从笔记纸外部打开临时文件后，会在外部编辑器保存时同步回写纸片内容。
- **胶囊模式**：支持普通折叠胶囊、右侧贴边胶囊、悬停滑出、拖拽重排和主胶囊一键收纳。
- **胶囊动效和视觉**：参考原 Windows 版演示动图，优化胶囊尺寸、贴边露出宽度、堆叠间距、主胶囊文案和滑出动画节奏。
- **胶囊状态稳固**：减少隐藏纸片、关闭贴边模式、修改标题或隐藏已关联笔记后出现错位、残留占位和形状异常的情况。
- **贴边展开优化**：贴边胶囊打开纸片后会从侧边消失，展开位置贴近屏幕右侧并避开边缘槽位，同时避免普通纸片位置被误保存。
- **待办关联笔记**：支持将笔记纸标题拖到待办项上建立关联，可从待办行快速打开或取消关联，并可按设置显示关联笔记名称。
- **标题与右键菜单**：支持双击纸片标题重命名，并补齐纸片和待办项的常用右键操作。
- **主题和配色**：支持跟随系统、浅色、深色，以及暖纸、墨、林、霞四套配色。
- **多语言界面**：菜单栏、设置、弹窗、待办操作、Markdown 快捷占位和胶囊提示支持中文、英文、日文、韩文，并默认跟随系统语言。
- **桌面空间策略**：设置中新增“在所有桌面显示纸片”，可选择纸片和胶囊是否跨 Spaces 显示。
- **开机自启动**：支持在设置中启用登录后自动启动，并在需要系统批准或注册失败时给出提示。
- **单实例与启动参数**：支持重复启动时转发显示、隐藏、切换、新建待办、新建笔记、导入数据和退出命令。
- **数据导入**：支持从菜单栏或启动参数导入原 Windows 版 `data.json`，导入失败不会修改当前数据。
- **数据保护增强**：读取旧数据或缺少新增字段时更稳健，避免加载失败后用空状态覆盖原有数据。
- **中文输入修复**：修复中文输入法组合输入时可能把拼音字母一并写入待办或笔记的问题。
- **纸片视觉优化**：收紧顶栏、正文、复选框和底部追加区的比例，并补齐轻量顶栏分隔线。
- **设置说明补齐**：设置页补齐各项悬停说明，并保留启动项、胶囊、顶栏和待办关联选项的扩展提示。
- **未公证发布包**：新增开源分发用的未公证 `.app.zip` 打包流程，并随包生成校验和与首次运行说明。
